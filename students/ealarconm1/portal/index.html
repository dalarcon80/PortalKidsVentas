La Tienda de Bloques

Portal de Misiones

Historia

La Tienda de Bloques necesita dos Analistas Jóvenes: Ventas y Operaciones. Su misión es construir un sistema simple que lea pedidos, los limpie y prepare reportes. Aprenderán paso a paso con retos cortos, pistas y una Llave que desbloquea la siguiente misión.

Cómo funciona

  1. En el Explorador abre \\34.134.152.136\misiones\starter y ejecuta RunFirst.ps1.
  2. Eso copiará el proyecto a tu PC y abrirá la guía. Completa M1: instalar VS Code y luego Python.
  3. En la terminal escribe python src/quest.py enroll VENTAS GOLEM12 (o OPERACIONES GHAST77).
  4. Cuando pases una misión, escribe python src/quest.py next. El motor te mostrará un enlace con un código secreto para la siguiente misión.

Empezar: M1 — La Puerta de la Base

¿Te atoras? Busca la página de pistas de tu misión.

¿Ya desbloqueaste una misión?

Usa el enlace que te imprime el motor. Se verá así: http://34.134.152.136/portal/<HASH>/index.html.
