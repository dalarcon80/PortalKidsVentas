M1 — La Puerta de la Base (Instalar VS Code y preparar Python)

Objetivo: Tener listo tu taller (VS Code) y probar que funciona, y luego instalar Python.

¿Por qué importa?

VS Code es donde vas a escribir y ejecutar tus programas. Sin él no puedes avanzar. Python es el idioma que hablará tu computadora.

Qué investigar (anota en docs/m1_notes.md)

  • ¿Qué es un editor de código? Busca un video corto sobre VS Code.
  • ¿Qué es un intérprete de Python? ¿Qué significa “agregar a PATH”?
  • ¿Qué es una “terminal” y para qué sirve?

Pasos

  1. Instala VS Code para Windows 11 (marca Add to PATH si aparece).
  2. Abre “Terminal” dentro de VS Code.
  3. Prueba y guarda evidencia (se crea un archivo de texto):
    code --version > docs/m1_vscode_ok.txt
  4. Instala Python (Windows x64). Al final, prueba:
    python --version
  5. Valida M1 y pide tu Llave:
    python src/quest.py run M1
    python src/quest.py next
